# ToDoList_API

Веб-приложение, которое предоставляет RESTful API для управления списком задач (TODO list).

## Использованные технологии:
+ Flask
+ SQLAlchemy
+ Swagger_ui

## Функции:
+ Создать новую задачу
+ Обновить существующую задачу
+ Удалить существующую задачу
+ Показать существующую задачу
+ Показать список всех задач

## Запуск приложения:
1. Установить необходимые пакеты, выполнив команду pip install -r requiremens.txt
2. Указать данные для доступа к базе данных в файле config.py
3. Запустить main.py

Документация доступна по адресу /api/docs.

Для проверки корректности работы API используется файл test.py.
